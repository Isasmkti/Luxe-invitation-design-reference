# Digital Invitation Landing Page

Landing page sederhana untuk **katalog undangan digital** dengan tampilan **elegan tema emas & putih**.
Dibuat menggunakan **HTML dan Tailwind CSS**, **tanpa JavaScript**.

---

## Teknologi

* HTML5
* Tailwind CSS (CDN)

---

## Fitur

* Landing page statis
* Katalog produk undangan digital
* Tampilan grid card
* Tombol **Live Demo**
* Tombol **Pesan via WhatsApp**
* Desain elegan (gold & white)
* Responsive

> Filter dan pagination dibuat **secara visual (static)** tanpa logika JavaScript.

---

## Struktur File

```bash
project/
├── index.html
└── README.md
```

---

## Cara Menjalankan

1. Buka file `index.html`
2. Tidak perlu build atau server
3. Bisa langsung dibuka di browser

---

## Tailwind CDN

Tailwind digunakan via CDN:

```html
<script src="https://cdn.tailwindcss.com"></script>
```

---

## WhatsApp Order

Gunakan format link berikut pada tombol pesan:

```html
https://wa.me/628xxxxxxxxxx
```

Ganti nomor sesuai kebutuhan.
